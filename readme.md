# Vapeur - Plateforme e-commerce de jeux vidéo

## Présentation du projet

**Vapeur** est une plateforme e-commerce de jeux vidéo inspirée de Steam, permettant aux utilisateurs de parcourir, acheter et gérer des jeux dématérialisés, tout en profitant d’un espace communautaire interactif.

L’objectif principal est de proposer une application complète où chaque utilisateur peut explorer une librairie de jeux, gérer sa bibliothèque personnelle et interagir avec ses contacts.

---

## Objectifs de l'application

- Permettre aux utilisateurs de parcourir une librairie de jeux dématérialisés.
- Proposer un espace personnel riche avec profil, bibliothèque et inventaire.
- Faciliter l’aspect communautaire via les contacts, discussions et contenus partagés.
- Offrir des fonctionnalités avancées de filtrage, personnalisation et suivi des succès.

---

## Fonctionnalités principales

### Magasin

- Parcourir le catalogue de jeux
- Liste des découvertes basée sur les jeux possédés
- Liste de souhaits
- Boutique de points
- Actualités et classements
- Afficher le détail d’un jeu :
   - Titre
   - Courte description
   - Prix
   - Date de publication
   - Avis des autres utilisateurs
- Acheter un jeu
- Vérification de l’âge selon les catégories

### Bibliothèque personnelle

- Liste des titres possédés
- Titres actuellement installés sur la machine
- Titres en commun avec les contacts
- Accès aux succès déverrouillés et déverrouillables
- Gestion de l’inventaire

### Profil utilisateur

- Pseudo et tag (#) de l’utilisateur
- Biographie
- Liste des titres mis en avant
- Liste des titres les plus joués
- Liste des succès liés aux jeux possédés
- Accès à la liste de contacts

### Communauté

- Participer aux discussions
- Accéder au Workshop et au Marché
- Visionner des diffusions
- Rejoindre des groupes et communautés

## Fonctionnalités détaillées

- Afficher le catalogue complet des jeux
- Filtrer par catégorie(s)
- Accéder au panier
- Accéder à la liste de souhaits
- Accéder à la page de profil
- Accéder à la liste de contacts
- Vérification de l’âge selon les catégories de jeu
- Liste des communautés par jeu
- Rédaction et consultation des avis pour les jeux
- Accès à sa bibliothèque personnelle
- Suivi des succès déverrouillés et déverrouillables
- Gestion et consultation de l’inventaire

---

## Technologies et outils

- Frontend : React / React Native
- Backend : Node.js + Express
- Base de données : PostgreSQL / MongoDB
- Diagrammes et schémas : PlantUML, Figma
- Gestion de versions : Git + GitHub
- Sécurité : HTTPS, OAuth2, JWT
- Méthodologie : Agile Scrum avec sprints de 2 semaines
